# 事实库 UI 重构计划：适配 67 卷 + 三层事实

## 当前问题

| 问题 | 影响 |
|------|------|
| 设定标签无卷筛选（第175行 `subTab !== 'setting'` 隐藏了卷栏） | 67卷时无法按卷查看设定 |
| 伏笔列表无卷级回收率 | 不知道哪卷的伏笔回收情况最差 |
| 时间线跳跃检测跨所有卷 | 67卷的时间跳跃混在一起无法定位 |
| 设定标签不区分全局/卷级 | 全局规则和卷专属设定混在一起 |
| factsByCh 用 established_chapter | 无法按 volume_number 分组显示 |
| 没有 chapter_deltas 的 UI | 用户看不到每章的实时变化 |

## 重构方案

### 1. 设定标签：三层视图

```
┌─────────────────────────────────────────┐
│ [角色][世界观][规则][关系][事件]          │  ← 分类不变
│                                         │
│ 卷筛选: [全部] [全局] [第1卷] [第2卷]...  │  ← 新增！水平滚动
│                                         │
│ 副筛选: [全部] [硬规则] [软规则]          │  ← 保留
│                                         │
│ ┌─ 全局设定 ─────────────────────────┐  │
│ │ 角色 程澈  ...                       │  │
│ │ 设定 世界观 ...                       │  │
│ └────────────────────────────────────┘  │
│ ┌─ 第2卷设定 ────────────────────────┐  │
│ │ 角色 凌瑶(本卷) ...                   │  │
│ └────────────────────────────────────┘  │
│                                         │
│ [+ 添加]  从大纲/卷纲提取                 │
└─────────────────────────────────────────┘
```

**改动**：
- `subTab === 'setting'` 时显示卷筛选栏（之前被隐藏）
- 事实列表按 `volume_number` 分组（全局在顶，当前卷紧随）
- 新增卷筛选 dropdown/buttons
- 新增事实时自动带 `volume_number`

### 2. 伏笔标签：卷级回收率 + 卷筛选

```
┌─────────────────────────────────────────┐
│ 卷筛选: [全部] [第1卷] [第2卷]...        │  ← 新增
│                                         │
│ ┌─ 全部回收率 ────────────────────────┐ │
│ │ ████████░░ 70%                       │ │
│ │ 已收 35 | 活跃 10 | 逾期 5 | 总计 50  │ │
│ └────────────────────────────────────┘  │
│                                         │
│ ┌─ 第2卷回收率 ──────────────────────┐  │  ← 新增！per-volume
│ │ ████░░░░░░ 40%                       │ │
│ └────────────────────────────────────┘  │
│                                         │
│ [按卷章展开] ...                         │  ← 保留，但按筛选卷过滤
└─────────────────────────────────────────┘
```

**改动**：
- 新增 `getForeshadowVolumeStats(projectId, volumeNumber)` 函数
- 标题栏加卷筛选 dropdown
- 按卷过滤伏笔列表

### 3. 时间线标签：卷级跳跃 + 卷筛选

```
┌─────────────────────────────────────────┐
│ 卷筛选: [全部] [第1卷] [第2卷]...        │  ← 新增
│                                         │
│ ┌─ 第2卷时间跨度 ────────────────────┐  │
│ │ 第15天 → 第28天 (13天) 12个事件      │  │
│ │ ⚠ 1处跳跃: 第15章 +7天              │  │  ← 只显示当前卷的跳跃
│ └────────────────────────────────────┘  │
│                                         │
│ [按卷章展开] ...                         │  ← 保留
└─────────────────────────────────────────┘
```

**改动**：
- 新增 `getTimelineVolumeStats(projectId, volumeNumber)` 函数
- `detectTimeJumps` 增加 volume_number 参数
- 标题栏加卷筛选

### 4. 冲突标签：不改（量少）

冲突数量通常不多（设定矛盾检测），不需要卷筛选。但可以加统计：哪卷冲突最多。

### 5. 新增：章级 delta 子标签

```
子标签: [设定] [时间线] [伏笔] [动态] [冲突]  ← 新增「动态」

动态标签内容:
┌─────────────────────────────────────────┐
│ 卷筛选: [第2卷]...                       │
│                                         │
│ ┌─ 第5章 动态 ────────────────────────┐ │
│ │ 程澈: 情绪 警惕→紧张                  │ │
│ │ 程澈: 位置 教学楼→地下实验室B2          │ │
│ │ 程澈→凌瑶: 关系 陌生人→临时盟友         │ │
│ │ 凌瑶: 揭示「暗门是父亲设计的」          │ │
│ └────────────────────────────────────┘  │
│ ┌─ 第4章 动态 ────────────────────────┐ │
│ │ ...                                  │ │
│ └────────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

**改动**：
- 新增 `SubTab` 类型 `'deltas'`
- 新增 `canonService.getChapterDeltas(projectId, volumeNumber?)` 函数
- 按章节分组展示，每条显示类型图标
