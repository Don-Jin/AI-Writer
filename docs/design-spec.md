# UI 设计规范

## 设计原则

- **简洁**：减少视觉噪音，每个页面只展示必要元素
- **直观**：按钮和操作的语义清晰，无需学习即可上手
- **中文优先**：所有界面文字使用中文

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#FFFFFF` | 纯白背景 |
| 次背景 | `#F8F9FA` | 浅灰背景（用于卡片、编辑区） |
| 侧边栏背景 | `#F0F1F3` | 略深于主背景，区分导航区 |
| 分割线 | `#E5E7EB` | 轻量分割 |
| 主文字 | `#1F2937` | 深灰，保证可读性 |
| 次文字 | `#6B7280` | 中灰，用于辅助信息 |
| 强调色 | `#4A90D9` | 柔和蓝，按钮/链接 |
| 强调色悬停 | `#3A7BC8` | 深一度的蓝 |
| 成功 | `#10B981` | 绿色，成功提示 |
| 警告 | `#F59E0B` | 黄色，警告提示 |
| 错误 | `#EF4444` | 红色，错误提示 |

## 字体

| 用途 | 字体 | 大小 | 字重 |
|------|------|------|------|
| 页面标题 | 微软雅黑 / Microsoft YaHei | 24px | 600 (SemiBold) |
| 区块标题 | 微软雅黑 | 18px | 500 (Medium) |
| 正文 | 微软雅黑 | 15px | 400 (Regular) |
| 辅助文字 | 微软雅黑 | 13px | 400 |
| 编辑器内容 | 微软雅黑 / 等宽回退 | 16px | 400 |

## 布局规范

### 整体布局

```
┌──────┬─────────────────────────────────────┐
│      │         顶部标题栏 (48px)              │
│ 侧   ├─────────────────────────────────────┤
│ 边   │                                     │
│ 栏   │         主内容区                      │
│      │       (白色/浅灰背景)                  │
│ 200  │                                     │
│ px   │                                     │
│      │                                     │
│      │                                     │
└──────┴─────────────────────────────────────┘
```

- 窗口默认尺寸：1200 × 800 px
- 最小窗口尺寸：900 × 600 px
- 侧边栏宽度：200px，不可拖拽调整
- 主内容区内边距：32px

### 侧边栏导航

```
┌────────────┐
│  📖 AI写作  │  ← Logo/标题
├────────────┤
│  🏠 首页    │  ← 项目列表
│  📚 风格库  │  ← 风格库管理
│  ⚙  设置    │  ← 系统设置
│            │
│            │
│            │
│            │
│  v1.0.0    │  ← 版本号（底部）
└────────────┘
```

- 导航项高度：44px
- 激活项：左侧 3px 蓝色边框 + 浅蓝背景 (#EBF3FC)
- 图标使用 emoji 或简单 SVG

### 间距规范

- 组件间间距：16px / 24px
- 卡片内边距：20px
- 按钮内边距：8px 16px（中）/ 6px 12px（小）
- 输入框内边距：10px 14px
- 圆角：8px（卡片）、6px（按钮/输入框）

## 组件规范

### 按钮

| 类型 | 背景 | 文字 | 边框 |
|------|------|------|------|
| 主按钮 | `#4A90D9` | 白色 | 无 |
| 次按钮 | 白色 | `#4A90D9` | `#4A90D9` |
| 危险按钮 | 白色 | `#EF4444` | `#EF4444` |
| 禁用 | `#E5E7EB` | `#9CA3AF` | 无 |

### 输入框

- 边框：1px solid `#D1D5DB`
- 聚焦时：边框变为 `#4A90D9`，外发光 `0 0 0 3px rgba(74,144,217,0.2)`
- 高度：40px（单行）/ 自适应（多行）
- Placeholder 颜色：`#9CA3AF`

### 卡片

- 背景：`#FFFFFF`
- 边框：1px solid `#E5E7EB`
- 圆角：8px
- 阴影（悬停时）：`0 2px 8px rgba(0,0,0,0.08)`

### 加载状态

- API 调用期间显示骨架屏或加载动画
- 生成文本时显示逐字输出效果（可选）
- 任何阻塞操作显示不可关闭的进度提示

## 页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 项目列表（首页） | 展示所有小说项目 |
| `/library` | 风格库管理 | 展示风格库列表 |
| `/project/:id/outline` | 大纲编辑 | 生成+编辑大纲 |
| `/project/:id/detail-outline` | 细纲编辑 | 生成+编辑细纲 |
| `/project/:id/write` | 章节写作 | 逐章生成+编辑 |
| `/settings` | 设置 | API Key 等配置 |
